# Telegram Live Profile

Скрипт для автоматического обновления имени профиля в Telegram на текущее время в реальном времени.

## Описание

Этот Python-скрипт использует библиотеку Telethon для автоматического обновления имени вашего профиля в Telegram. Имя профиля меняется каждую минуту и отображает текущее время в формате `ЧЧ:ММ` с использованием стилизованных цифр.

## Особенности

- Автоматическое обновление имени профиля каждую минуту
- Стилизованные цифры времени (используются жирные символы)
- Обработка ошибок (FloodWait, проблемы с соединением)
- Логирование важных событий
- Работа в фоновом режиме
- Использование Telethon для взаимодействия с Telegram API
- Кастомизированные символы Unicode для стилизации времени
- Минимальная нагрузка на систему

## Деплой на VPS

1. Добавьте GitHub Secrets: Settings → Secrets and variables → Actions. Обязательно добавьте:

```secrets
VPS_HOST = адрес сервера
VPS_USER = SSH пользователь
VPS_SSH_KEY = файл приватного ключа
VPS_APP_PATH = путь деплоя на VPS
API_ID = ваш telegram API ID
API_HASH = ваш telegram API hash
SESSION_STRING = ваша telethon session
VPS_SSH_PORT = номер порта SSH, если не 22
```

2. Cделайте push в main:

```bash
git push origin main
```
